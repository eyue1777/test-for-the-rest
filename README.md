## 📊Task 1 – Number Analysis App

 A number analysis program that calculates the maximum, minimum, sum, and average of a list of integers.

Features include:

- Finding the maximum value
- Finding the minimum value
- Calculating the sum
- Calculating the average

Additional (Bonus) features implemented:
```
✅ Empty list validation
✅ Counting negative numbers
✅ Sorting the list without using `.sort()`
✅ Comparison with Dart collection methods
```
```
Task 1 Sample Output:
Numbers: [34, -7, 89, 12, -45, 67, 3, 100, -2, 55]
Max: 100
Min: -45
Sum: 306
Average: 30.6
```

## ⚡Task 2 – Async Calculator App

An asynchronous calculator app demonstrating Futures, async/await, and error handling.

Features include:

- Calculator class with arithmetic operations
- Asynchronous calculations using `Future`, `async`, and `await`
- Error handling with `try-catch`

Additional (Bonus) features implemented: 
```
✅ Custom exception for invalid operations
✅ Calculation history
✅ Chained operations
✅ Parallel execution using `Future.wait()`
```
